sequenceDiagram
    participant C as Client<br/>(mcp-client.sh)
    participant KC as Keycloak<br/>(OIDC Provider)
    participant E as Envoy Proxy<br/>(gateway-system)
    participant R as Router<br/>(ext_proc, mcp-system)
    participant A as Authorino<br/>(Wasm plugin → kuadrant-system)
    participant B as Broker<br/>(mcp-system)
    participant S as MCP Server<br/>(mcp-test)
    Note over C,S: 1. Token Acquisition (Device Authorization Grant)
    C->>KC: POST /auth/device<br/>client_id=mcp-cli&scope=openid groups
    KC-->>C: device_code, user_code ABCD-EFGH, verification_uri
    Note over C: Opens browser → user logs in
    C->>KC: POST /token<br/>grant_type=device_code
    KC-->>C: access_token, id_token, refresh_token
    Note over C: Token contains: sub, groups, iss
    Note over C,S: 2. Authenticated Request (HTTP or HTTPS)
    alt HTTPS (port 8443)
        Note over C,E: TLS handshake using cert signed by mcp-ca-issuer
        C->>E: POST https://.../mcp Authorization: Bearer eyJ...
        Note over E: TLS terminated at Gateway
    else HTTP (port 8001)
        C->>E: POST http://.../mcp Authorization: Bearer eyJ...
    end
    Note over E: Filter chain: ext_proc → Wasm → router
    E->>R: ext_proc: headers + body
    alt initialize / tools/list (aggregation path)
        Note over C,S: 3a. Initialize or tools/list → Broker aggregates
        R-->>E: authority unchanged, x-mcp-method set
        E->>A: CheckRequest (host, path, JWT)
        Note over A: Gateway AuthPolicy: Validate JWT via JWKS
        A-->>E: OK (authorized)
        alt Missing/invalid token
            A-->>E: DENIED
            E-->>C: 401 Unauthorized
        end
        E->>B: POST /mcp tools/list
        B->>S: POST /mcp → tools/list
        S-->>B: tools response
        B-->>E: Aggregated tools
        E-->>C: Aggregated tools/list response
    else tools/call (direct routing path)
        Note over C,S: 3b. tools/call → Router + AuthPolicy + direct to server
        R-->>E: authority=test-server1.mcp.local, x-mcp-toolname=greet
        E->>A: CheckRequest (host, toolname, JWT)
        Note over A: Per-HTTPRoute AuthPolicy: group + tool ACL check
        alt Authorized
            A-->>E: OK
            E->>S: POST /mcp tools/call greet
            S-->>E: tool result
            E-->>C: 200 OK + tool result
        else Unauthorized
            A-->>E: DENIED
            E-->>C: 403 Forbidden
        end
    end
